# 📁 SQL-Assignment

This folder contains hands-on SQL assignments designed to strengthen database concepts through practical implementation.

Instead of focusing only on theory, every topic includes executable SQL scripts that help understand how SQL Server works in real-world scenarios.

---

## 📚 Topics Covered

### 🔹 Joins

Combine data from multiple tables using relationships.

Common joins included:

- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- SELF JOIN
- CROSS JOIN

> Joins allow relational databases to retrieve meaningful information spread across multiple tables.

---

### 🔹 Cursor

A Cursor processes query results one row at a time instead of all rows together.

Useful when:

- Row-by-row processing is required
- Performing sequential operations
- Complex business logic

Although powerful, cursors should only be used when set-based operations are not suitable.

---

### 🔹 Stored Procedures

Stored Procedures are reusable collections of SQL statements stored inside the database.

Benefits include:

- Code reusability
- Better performance
- Reduced network traffic
- Improved security
- Easier maintenance

---

### 🔹 Functions

Functions return a value after performing calculations or processing data.

Examples include:

- Scalar Functions
- Table-Valued Functions

Functions are commonly used for:

- Calculations
- Data formatting
- Reusable business logic

---

### 🔹 Triggers

Triggers execute automatically when specific database events occur.

Supported events include:

- INSERT
- UPDATE
- DELETE

Common use cases:

- Audit logging
- Data validation
- Automatic updates
- Business rule enforcement

---

## 🎯 Purpose

These assignments help in understanding:

- Relational database concepts
- Query writing
- SQL Server programming
- Database automation
- Performance-oriented SQL coding
